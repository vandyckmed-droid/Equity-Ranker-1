PATCH 5 — METHODOLOGY / DOCUMENTATION UPDATE
Update the human-facing methodology and AI notes to clearly define and distinguish the following terms in simple mathematical language:

- variance
- covariance
- correlation
- covariance matrix
- correlation matrix
- portfolio variance

Requirements:
- explain that variance is for a single asset’s dispersion
- explain that covariance describes how two assets move together in raw units
- explain that correlation is normalized covariance and ranges from -1 to +1
- explain that variance is a special case of covariance: Var(X) = Cov(X, X)
- explain that portfolio variance is computed from the covariance matrix:
  portfolio variance = w'Σw
- explain that correlation matrix is useful for similarity, clustering, and “names that move together”
- explain that correlation matrix alone is not sufficient for true portfolio variance unless combined with asset volatilities
- keep this explanation concise, plain, auditable, and math-focused rather than code-focused
